Небольшое веб-приложение на `Flask + SQLite` для хранения контактов.

## Что реализовано

- список контактов
- добавление нового контакта
- редактирование контакта
- удаление контакта
- поиск по имени, фамилии или телефону
- сортировка по фамилии
- страница с подробной информацией о контакте
- базовая валидация ввода

## Используемые технологии

- Python
- Flask
- SQLite
- Jinja2

## Структура проекта

- `app.py` - основной файл приложения
- `schema.sql` - схема базы данных
- `templates/` - HTML-шаблоны
- `static/style.css` - стили сайта

## Как запустить

1. Открыть PowerShell в папке проекта.
2. Перейти в папку приложения:

```powershell
cd flask_contacts_11
```

3. Создать виртуальное окружение:

```powershell
python -m venv venv
```

4. Активировать виртуальное окружение:

```powershell
.\venv\Scripts\Activate.ps1
```

5. Установить Flask:

```powershell
python -m pip install flask
```

6. Запустить приложение:

```powershell
python app.py
```

7. Открыть в браузере:

```text
http://127.0.0.1:5000
```
